# 精神状态鉴定中心 - 重设计方案

## 项目概述

对现有"精神状态鉴定中心"SBTI风格人格测试进行全面重设计，包括：
- 视觉风格升级（赛博故障美学 → 深紫色现代感设计）
- 内容扩充（20题 → 40题）
- 人格原型重构（16个 → 20个，鲁迅黑色幽默风格）
- 评分系统优化（二元选择 → 4-6选项混合评分）
- 结果展示丰富化（雷达图 + 进度条 + 详细分析）

## 设计目标

1. **社会价值**：每个原型代表一种社会应对策略，帮助参与者反思自己的应对方式
2. **鲁迅风格**：黑色幽默贯穿整个测试过程，讽刺、自嘲、反讽
3. **量化分析**：多维度评分系统，提供详细的精神状态分析
4. **视觉升级**：深紫色/深蓝色调，渐变光晕，现代卡片式布局

## 维度设计

### 四个核心维度

| 维度 | 极点A | 极点B | 社会意义 |
|------|-------|-------|----------|
| 压力应对 | 抗争 | 逃避 | 面对社会压力的选择 |
| 情绪管理 | 压抑 | 宣泄 | 处理情绪的方式 |
| 社交模式 | 疏离 | 连接 | 与他人建立关系的方式 |
| 决策方式 | 理性 | 感性 | 做决定的依据 |

### 维度得分计算

每个维度的得分范围：-20 到 +20（每题最高±2分，40题）

最终每个维度生成一个0-100的指数：
```
维度指数 = (得分 + 20) / 40 * 100
```

## 题目设计

### 题目数量

- 总计40题
- 每维度10题
- 随机打乱顺序

### 选项设计

- 每题4-6个选项
- 选项含义不定，可以是各种社会应对策略
- 鲁迅式黑色幽默风格

### 评分机制（混合模式）

#### 选项-维度映射表（关键选项）

每个选项有明确的维度和分值：
```javascript
const optionScoring = {
  // 格式：选项文本 → { dimension: 分值 }
  "立刻回复，然后通宵完成任务": { stress: 2, emotion: 0, social: 0, decision: 0 },
  "假装没看见，明天再说": { stress: -2, emotion: 0, social: 0, decision: 0 },
  // ...
};
```

#### 选项语义分类（其他选项）

将选项按语义分为几类，自动计算维度得分：
```javascript
const semanticCategories = {
  "积极应对": { stress: 1, emotion: 0, social: 0, decision: 0 },
  "消极应对": { stress: -1, emotion: 0, social: 0, decision: 0 },
  "情绪化应对": { stress: 0, emotion: 1, social: 0, decision: 0 },
  // ...
};
```

### 题目示例

#### 短选项题目

**题目1（压力应对维度）：**

你的老板在凌晨三点发了一条消息："在吗？"你的第一反应是？

- A: 立刻回复，然后通宵完成任务（抗争 +2）
- B: 假装没看见，明天再说（逃避 +2）
- C: 回复"在的"，然后继续睡觉（中立 +0）
- D: 把老板拉黑，然后继续睡觉（极端逃避 +3）

#### 长选项题目

**题目6（压力应对维度）：**

你正在赶一个重要的项目截止日期，突然发现数据出了问题，需要重做。此时你的内心独白最接近哪种？

- A: "完了完了完了，这次死定了。我应该早点检查的，为什么不早点检查？我现在该怎么办？如果搞砸了，老板会怎么看我？同事会怎么看我？我是不是不适合这份工作？"（内心崩溃型 - 抗争+1，压抑+2）

- B: "操，又是这样。每次都这样。行吧，重做就重做。反正我的命就是给公司擦屁股的。老子不干了。"（愤怒宣泄型 - 抗争+2，宣泄+2）

- C: "好的，数据出了问题。让我先深呼吸，然后列一个计划：第一步，确认问题范围；第二步，评估补救方案；第三步，执行修复；第四步，汇报进度。现在开始第一步。"（冷静应对型 - 抗争+1，理性+2）

- D: "这个问题...其实也没那么重要吧？反正老板也不会仔细看。我随便改改就好了。deadline是死的，人是活的。"（自我安慰型 - 逃避+2，宣泄-1）

- E: "我需要帮助。我现在应该找谁？小王之前遇到过类似的问题，我可以问他。或者找老板说明情况，看看能不能延期。我不能一个人扛。"（寻求支持型 - 抗争+0，连接+2）

#### 精神状态评分

**核心维度评分（各维度0-100分）：**

| 维度 | 评分范围 | 评分说明 |
|------|----------|----------|
| 压力应对指数 | 0-100 | 0=完全逃避，100=完全抗争 |
| 情绪管理指数 | 0-100 | 0=完全压抑，100=完全宣泄 |
| 社交模式指数 | 0-100 | 0=完全疏离，100=完全连接 |
| 决策方式指数 | 0-100 | 0=完全理性，100=完全感性 |

**综合指数评分（0-100分）：**

| 指数名称 | 计算方式 | 社会意义 |
|----------|----------|----------|
| 社会适应度 | 综合4个维度的得分，考虑社会主流价值观 | 你在当前社会中的适应程度 |
| 心理健康度 | 综合考虑情绪管理、压力应对等因素 | 你的心理健康状态 |
| 情绪稳定性 | 考虑情绪管理、压力应对的平衡性 | 你的情绪波动程度 |
| 社交健康度 | 考虑社交模式、情绪管理的平衡性 | 你的人际关系质量 |
| 决策健康度 | 考虑决策方式、压力应对的平衡性 | 你的决策质量 |

**综合精神状态评分（0-100分）：**

计算公式：
```
综合评分 = (社会适应度 × 0.3) + (心理健康度 × 0.3) + (情绪稳定性 × 0.2) + (社交健康度 × 0.1) + (决策健康度 × 0.1)
```

评分等级：

| 评分范围 | 等级 | 描述 |
|----------|------|------|
| 90-100 | 圣人级 | 你已经超越了世俗的烦恼，达到了精神自由的境界 |
| 80-89 | 人间清醒 | 你看透了世界的荒诞，但依然选择温柔地活着 |
| 70-79 | 正常人类 | 你和大多数人一样，在崩溃和自愈之间反复横跳 |
| 60-69 | 轻微内耗 | 你有一些精神内耗，但还能维持基本功能 |
| 50-59 | 中度内耗 | 你的精神状态有点糟，需要关注一下 |
| 40-49 | 重度内耗 | 你已经在崩溃的边缘，建议寻求帮助 |
| 30-39 | 精神濒危 | 你的精神状态需要紧急干预 |
| 0-29 | 已死但还在呼吸 | 你的精神已经死亡，但肉体还在打卡 |
