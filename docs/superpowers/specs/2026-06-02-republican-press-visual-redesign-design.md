# 民国报章 / 鲁迅文学风视觉重设计

- **日期**：2026-06-02
- **范围**：`index.html` 的 `<style>` 块、canvas 背景动画、三个页面（intro / question / result）的视觉呈现
- **不改动**：题目数据、原型数据、评分逻辑、页面切换逻辑、DOM 结构（除非视觉需要）

## 背景与动机

当前视觉是「深紫赛博故障美学」（霓虹紫蓝渐变、RGB 错位 glitch、二进制雨、骷髅抖动）。用户对现有风格不满意，希望换成更贴合项目精神的风格。

项目精神：**精神状态鉴定中心 · SBTI 风格抽象人格测试 · 鲁迅式黑色幽默文案**。

经过三套方向（临床档案 / 故障终端 / 民国报章）的可视化对比，用户选定 **民国报章 / 鲁迅文学风**——把项目里「鲁迅」这条文化线做实，文学味最浓，与黑色幽默文案的设定最贴合。

## 设计决策（已与用户确认）

| 维度 | 决策 | 说明 |
|------|------|------|
| **文案处理** | 视觉民国风，正文文案保持简体白话 | 题目、选项、原型描述等正文内容**一字不改**，仅在标题/标签/装饰位用文言点缀（如「問」「鑒定」印章、旧报口吻装饰句）。兼顾调性与可读性。 |
| **背景动画** | 保留 canvas 动画，二进制雨 → 墨点/落字雨 | 不删除 canvas，改造粒子内容为民国调子（飘落的墨点、纵向汉字雨），保持动态生命力但不与纸张风冲突。 |
| **力度** | 浓郁民国风 | 纸色、宋体、朱红印章、双线分隔都做足，饱和的旧报观感，追求记忆点。 |

## 视觉规范

### 配色（CSS 变量）

```
--paper:        #e9e2d0   泛黄纸色（主背景）
--paper-dark:   #ddd2ba   纸色阴影面
--ink:          #1f1810   墨黑（正文）
--ink-soft:     #4a3f30   淡墨（次要文字）
--sepia:        #8a6b3a   赭黄（标签、装饰）
--vermilion:    #a8321f   朱红（印章、强调、按钮）
--vermilion-dim:#7a2415   朱红暗部（投影）
--line:         #2b2620   分隔线
```

赛博风的紫/蓝/霓虹粉/绿全部移除。

### 字体

- 主字体：`Noto Serif SC`（宋体），权重 400/500/700/900
- 移除 `Courier New` 等宽标题字体
- 离线回退：系统 serif

### 关键视觉元素

1. **朱红印章**：右上角方形印章（「鑒定」/「已鑒」），竖排、轻微旋转、描边样式、半透明。
2. **双线分隔**：`border: 3px double` 朱墨色，用于标题下、结果区分隔。
3. **纸张质感**：泛黄底色 + 细点纹理（`radial-gradient` 点阵）+ 内阴影薄古渍（`inset box-shadow`）。
4. **圈号列项**：选项前缀用「〇」朱红圈号；题目前缀用「問」朱红字。
5. **进度条/指数条**：纸色轨道 + 朱红/赭黄填充，替换原赛博渐变。
6. **按钮**：朱红底 + 硬投影（`box-shadow: 3px 3px 0`），文言文案（「開始鑒定」）。

### 三屏布局（沿用现有结构，仅换皮）

- **intro（封面）**：印章 + 骷髅/笔符号（sepia 滤镜降饱和）+ 宋体大标题 + 双线 + 旧报口吻装饰句（dateline）+ 简体白话简介 + 朱红按钮。
- **question（答题）**：进度信息（第 N 問 / 共 40 問 + 维度名）+ 纸色进度条 + 「問」字题目 + 「〇」圈号选项（hover 右移变白）。正文全简体。
- **result（结果）**：「已鑒」印章 + 原型名大字 + typeCode + 双线 + 四维度纸色指数条 + 朱红引言区（鲁迅式哲言）。

### 背景动画改造

`canvas#bg-canvas` 保留，重写绘制逻辑：

- 二进制雨（1/0 字符）→ **墨点/汉字落雨**：随机汉字或墨点字符，墨色（`rgba(31,24,16,*)`），缓慢下落。
- 移除 glitch 方块、扫描线、RGB 噪点。
- 整体透明度降低，作为纸面上的淡墨背景，不抢正文。
- 保留现有节流优化思路，避免低性能设备卡顿。

## 实现注意事项（来自 CLAUDE.md gotchas）

- **CSS/视觉改动必须用 `frontend-design` skill**——本项目调性刻意，AI 默认样式会破坏。
- 改 `dimensions` 数组时需同步 `showResult()` 的 `allDimLabels` 和 `dimNames`——本次**不改维度**，但视觉调整结果页时注意不要误动这两处硬编码标签。
- Google Fonts 通过 `@import` 加载，需保留离线回退。
- Canvas 改造后注意性能（粒子数量、绘制频率）。

## 验收标准

1. 三个页面（intro/question/result）全部呈现浓郁民国旧报风，与定稿一致。
2. 正文（题目、选项、原型描述、简介）保持现有简体白话，未被改成文言。
3. canvas 背景为墨点/落字雨，无赛博 glitch 残留。
4. 赛博配色（紫/蓝/霓虹）全部清除，统一为纸/墨/朱红体系。
5. 离线环境字体正确回退，页面在浏览器直接打开即可运行（无构建步骤）。
6. 移动端响应式不破版。
